Help Document
This MiniCAD is developed by Tang Zihao in JAD (2020 Fall).
You can load / store .dat files to open / save the CAD file via File.Open / Save in the menu.
If you want to draw a shape, please click on the button the left and then click on the plain
area on the right.In case you want to change the color of one shape, single-click on the
palette on the left-down corner. If you want to change the default color, double-click on the
palette please.You are free to choose, drag, delete the shapes on the canvus.

Here are some detailed operations on the shapes:
Key Event
E Enlarge the shape
S Shrink the shape
+ Thicken the line (doesn't work to text)
- Thin the line (doesn't work to text)
↑ Move up the shape
↓ Move down the shape
← Move the shape to the left
→ Move the shape to the right
Backspace or Delete Delete the shape
